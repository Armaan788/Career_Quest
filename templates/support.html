Career Quest v1.0

Common Help Topics

  • If XP does not update, finish the quiz and refresh your dashboard.
  • If achievements do not appear, check that the career simulation was completed.
  • If saved careers do not show, make sure you are using the same player name.
  • If the career match page looks empty, retake the career match quiz.
  • If you want a fresh profile, use Switch Player from the home page.

Report A Problem

When reporting a problem, include what page you were on, what button you clicked, and what happened.
